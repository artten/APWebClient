Chat App by Artiom, Ilana and Saar

Dependencies:

1. node.js
2. npm
3. react-route-dome
4. react-bootstrap

How to run the app:

1. clone the repo
2. change directory to the project
3. install the dependencies
4. checkout to master branch
5. run the command "npm start" in the terminal
